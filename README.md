# FRONT WEB (APS)

Projeto desenvolvido para a disciplina de Desenvolvimento Front-End, com o objetivo de criar a camada cliente de uma aplicação de lista de tarefas (to-do list), integrada a um servidor já existente.

## Funcionalidades

* Cadastro de usuário
* Login com autenticação
* Criação de tarefas
* Visualização de tarefas
* Edição de tarefas
* Exclusão de tarefas

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Integração com API, utilizando Node.js e Express

## Integração

O front-end se comunica com o back-end por meio de requisições HTTP (API REST), utilizando autenticação via token para acesso às rotas protegidas.

## Estrutura do projeto

* `server/` → back-end fornecido pelo professor
* `client/` → front-end desenvolvido

## Como executar

1. Inicie o servidor (backend)
2. Abra os arquivos da pasta `client` no navegador
3. Realize login ou cadastro
4. Utilize o sistema de tarefas

## Objetivo

Aplicar conceitos de desenvolvimento web, integração entre front-end e back-end, e boas práticas de organização de código.
